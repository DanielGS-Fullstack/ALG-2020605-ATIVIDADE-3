# ALG-2020605-ATIVIDADE-3
Atividade professor Rafael Funções Crie as funções em um único código local SEM utilizar nenhuma variável global.

O aluno deve desenvolver a lista de exercício em linguagem C utilizando o DEV-C++.

Crie um repositório PUBLICO no github
Crie as funções em um único código local SEM utilizar nenhuma variável global.
1. Calcular H:m;s
2. isPar (verificar se o número é par, retorna true se é par e false se é impar)
3. isImpar ( verificar se o número é ímpar, retorna true se é ímpar, e false se é par)
4. media (calcula média entre dois números)
5. maior (retorna o maior valor de dois números)
6. menor (retorna o menor valor de dois números)
7. Publique os códigos no seu repositório GitHub
8. Preencha o questionário com o link do seu repositório.
